# Somativa 3 Front-End - _🚀 GalaxyTours_

Esta formativa de Front-End avaliará a capacidade de implementar media queries de forma eficaz, para tornar o site responsivo para diferentes dispositivos. Este projeto é sobre a GalaxyTours, a maior empresa de turismo espacial no mundo que possuia um site com seus serviços, entretanto, não era responsivo para outros dispositivos além de computador.

## ⌨️ Tecnologias utilizadas
* HTML5
* CSS3

## 🌐 Implementações
* **Formatação da página para tablets (largura máxima de 768px)**: Mudança de três colunas para duas
* **Formatação da página para mobile (largura máxima de 480px)**: Mudança de três colunas para uma
